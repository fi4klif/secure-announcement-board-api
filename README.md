# Дошка оголошень - Bulletin Board Application

Функціональний веб-застосунок "Дошка оголошень", де користувачи можуть переглядати список оголошень, шукати потрібні пропозиції, створювати власні оголошення та видаляти їх.

## Функціональність

- ✅ **Головна сторінка** - Список оголошень з пошуком по назві, сортуванням за датою та пагінацією (10 оголошень на сторінку)
- ✅ **Перегляд оголошення** - Детальна інформація про конкретне оголошення з кнопкою видалення
- ✅ **Створення оголошення** - Форма з валідацією полів (категорія, назва, опис, ціна, контакти)
- ✅ **Видалення** - Через JavaScript fetch з HTTP методом DELETE
- ✅ **Валідація даних** - Перевірка обов'язкових полів, мінімальної довжини, коректності ціни
- ✅ **Обробка помилок** - 404 та 500 сторінки

## Вимоги

- Node.js (v16 або новіше)
- npm або yarn

## Установка

1. Клонуйте або завантажте проект
2. Перейдіть до директорії проекту:

   ```bash
   cd HW2\ Node
   ```

3. Встановіть залежності:

   ```bash
   npm install
   ```

4. Ініціалізуйте базу даних:
   ```bash
   npx prisma migrate dev --name init
   ```

## Запуск

### Розробка

```bash
npm run dev
```

Сервер буде запущено на `http://localhost:3000` з автоматичним перезавантаженням при змінах.

### Production

```bash
npm start
```

## Маршрути

| Метод  | Маршрут              | Опис                                     |
| ------ | -------------------- | ---------------------------------------- |
| GET    | `/`                  | Головна сторінка зі списком оголошень    |
| GET    | `/announcements`     | Форма для створення оголошення           |
| POST   | `/announcements`     | Обробка створення оголошення             |
| GET    | `/announcements/:id` | Перегляд детальної інформації оголошення |
| DELETE | `/announcements/:id` | Видалення оголошення                     |

## Параметри запиту (GET /)

- `search` - Пошуковий запит по назві оголошення (без урахування регістру)
- `sort` - Порядок сортування: `newest` (за замовчуванням) або `oldest`
- `page` - Номер сторінки (за замовчуванням 1)

Приклади:

- `http://localhost:3000/?search=ноутбук`
- `http://localhost:3000/?sort=oldest&page=2`
- `http://localhost:3000/?search=послуги&sort=newest&page=1`

## Валідація

### На клієнті

- Обов'язкові поля (required)
- Мінімальна та максимальна довжина (minlength, maxlength)
- Мінімальна ціна (min="0")

### На сервері

- Назва: мінімум 5, максимум 100 символів
- Опис: мінимум 10 символів
- Контактна інформація: мінимум 5 символів
- Ціна: додатне число
- Категорія: одна з чотирьох дозволених

## Користування

### Переглядання оголошень

1. На головній сторінці видно всі оголошення у вигляді карток
2. Використовуйте поле пошуку для фільтрації за назвою
3. Виберіть порядок сортування з випадного списку
4. Натисніть "Детальніше" на картці для перегляду повної інформації

### Створення оголошення

1. Натисніть кнопку "+ Додати оголошення"
2. Заповніть всі обов'язкові поля форми
3. Натисніть "Опублікувати"
4. Вас буде перенаправлено на сторінку створеного оголошення

### Видалення оголошення

1. Відкрийте оголошення, натиснувши "Детальніше"
2. На сторінці деталей натисніть кнопку "Видалити"
3. Підтвердіть видалення у діалоговому вікні
4. Вас буде перенаправлено на головну сторінку

## Бази даних

Проект використовує SQLite з Prisma ORM. База даних зберігається у файлі `dev.db` в корневій директорії.

### Переглядання бази даних

```bash
npx prisma studio
```

Це відкриє веб-інтерфейс на `http://localhost:5555` для перегляду та управління даними.

## Технології

- **Express.js** - Веб-фреймворк
- **Prisma** - ORM для роботи з базою даних
- **SQLite** - База даних
- **EJS** - Шаблонізатор
